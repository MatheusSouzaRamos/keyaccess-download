# keyaccess-download

Sistema de acesso baseado em chave de acesso único.

KeyAccess Download é um sistema de download seguro de arquivos baseado em chaves de acesso temporárias e de uso único.
Após a autenticação bem-sucedida, o usuário é redirecionado para uma página de download protegida, inacessível por acesso direto via URL.
O sistema registra todas as tentativas de acesso, armazenando informações como usuário, horário, navegador e dispositivo, garantindo rastreabilidade e auditoria.



Este projeto está configurado para rodar no apache-tomcat
